# Проект - тестовое задание для hexlet

## Установка и запуск

1. Установить модули make install
2. Опубликовать модуль make publish
3. Установить модуль к себе npm link

Далее вы сможете использовать команды для игр:
1. brain-games
2. brain-even
3. brain-calc
4. brain-gcd
5. brain-progression
6. brain-prime
